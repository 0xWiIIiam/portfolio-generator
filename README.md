# Portfolio Generator

A simple web app to create and download your own portfolio website in seconds.

Built with HTML, CSS, and JavaScript. Users can create personalized portfolio websites with their profile picture, bio, social links, and projects.

---

## Features
* Live preview of website while editing
* Color picker to personalise the page
* Upload a profile picture
* Add multiple social links (GitHub, LinkedIn, Twitter, Instagram, Email)
* Add multiple projects with title, description, and optional links
* Responsive layout for mobile and desktop
* Download a ready to use HTML file
